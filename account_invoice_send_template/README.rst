This module extends the functionality of sending batch invoices and
allows to select the email template in the wizard before sending. If not
selected, it uses the default template.

.. IMPORTANT::
   This is an alpha version, the data model and design can change at any time without warning.
   Only for development or testing purpose, do not use in production.
   `More details on development status <https://odoo-community.org/page/development-status>`_

Usage
=====

To use this module, you need to:

1. Go to **Invoicing > Customers > Invoices**.
2. Select one or more invoices.
3. Click **Send** and, in the wizard, choose email template you want to
   use.
